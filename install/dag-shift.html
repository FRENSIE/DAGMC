Install DAG-Shift¶

This document explains how to install DAG-Shift, assuming you have already installed the required dependencies.

DAG-Shift is still under development at Oak Ridge National Laboratory. Shift is the massively parallel Monte Carlo code in Scale. DAG-Shift is internally functioning, but has not yet been released. For further information about DAG-Shift please contact Greg Davidson at ORNL.
